# Crawler Jobs

## Getting Started
### Tạo project nodejs
#### Khởi tạo project
    npm init

#### Tải thư viện puppeteer
    npm install -save puppeteer

#### Tải thư viện HCC
    npm i headless-chrome-crawler

## API 
* [HCC API](https://github.com/yujiosaka/headless-chrome-crawler)

* [Puppeteer API](https://pptr.dev/)
